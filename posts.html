Week 02

Some decision has been taken for the project. The general idea for the game is to compete throw the enemies. An AI system needs to be implemented for some enemies. This week I didn't work on the project, but instead I have been doing research in different types of techniques for the enemies.

Robocode was the the first approach to implement an agent system for the enemies. The main idea of Robocode is each agent has a set of instruction possible for each agent. The goal is to code an agent to survive in a battle.

Neuroevolution is a machine learning technique to evolve a neuronal network using genetics algorithm. It uses the output of the genetic algorithm for the backpropagation, and addind the possibility of cratating new neurons to improve the output of the aNN. In other words, uses the genetic algotihm to train the Neuronal Network

The main idea of Robocode its okay, but i don't like that I need to implement all the funcionts for all the possible functions and hardcode all the decisions. So I decided to use Neuroevolution instead. The algorithm for this is going to be NEAT (NeuroEvolution of Augmenting Topologies)

The main idea for the game is to start one run and the same time you compite against the machine. The goal is to see how deeper you can go without daying. At the same another thread is going to run the simulation to train the Network. Each time and the best enemy in the simulation is going to be choose to control the enemies.
